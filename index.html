Home Banking

Este es un simulador de home banking donde puedes realizar operaciones básicas como consultar saldo, depositar y retirar dinero.

El código define variables y constantes para almacenar el saldo y el nombre del usuario. Se crean funciones para mostrar el saldo, depositar dinero y retirar dinero. La función iniciarHomeBanking utiliza un bucle do...while y una estructura switch para manejar la interacción del usuario a través de cuadros de diálogo. La aplicación se inicia llamando a la función iniciarHomeBanking.
