Home Banking

Este es un simulador de home banking donde puedes realizar operaciones simples como consultar tu saldo, depositar dinero y retirar dinero.

El programa usa variables para guardar el saldo y el nombre del usuario. También tiene funciones que permiten realizar cada operación: mostrar el saldo, depositar dinero y retirar dinero.

El simulador funciona mostrando un menú de opciones al usuario. Para esto, se usa un bucle que repite el menú hasta que el usuario decide salir. Dependiendo de la opción elegida, se ejecuta una acción diferente.

La interacción se realiza con cuadros de diálogo (como prompt y alert) y también con mensajes que aparecen en la consola del navegador.
